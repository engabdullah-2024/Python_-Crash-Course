# Exercises: Syntax & Variables

Complete each exercise in its corresponding `exercise_N.py` file, then run it with `python exercise_N.py`. Solutions are in `solutions/`.

## Exercise 1 — Create and print variables (Easy)

File: `exercise_1.py`

Create three variables: `city` (a string), `population` (an int), and `is_capital` (a bool). Print all three.

## Exercise 2 — Fix the naming (Easy)

File: `exercise_2.py`

The starter code has poorly named variables (`x`, `y1`, `Z`). Rename them to descriptive `snake_case` names and update every place they're used, keeping the program's behavior identical.

## Exercise 3 — Reassignment practice (Medium)

File: `exercise_3.py`

Create a variable `status` set to the integer `1`. Print it and its type. Then reassign `status` to the string `"active"` and print it and its type again, to observe dynamic typing in action.

## Exercise 4 — Indentation debugging (Medium)

File: `exercise_4.py`

The starter file contains an `if` block with a deliberately broken/inconsistent indentation described in a comment. Fix the indentation so the code runs and produces the described output.

## Exercise 5 — Personal profile script (Hard)

File: `exercise_5.py`

Write a script that defines variables for `full_name`, `birth_year`, `favorite_language`, and `years_coding`, each with a comment explaining its purpose. Then print a short multi-line profile summary using several `print()` calls, and use `type()` to print the type of each variable.
