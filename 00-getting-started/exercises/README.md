# Exercises: Getting Started

Complete each exercise in its corresponding `exercise_N.py` file. Run your file with `python exercise_N.py` to check the output. Worked solutions are in `solutions/` if you get stuck — try each exercise yourself first.

## Exercise 1 — Print a greeting (Easy)

File: `exercise_1.py`

Write a program that prints `Hello, World!` to the terminal.

## Exercise 2 — Print multiple lines (Easy)

File: `exercise_2.py`

Print your name, your favorite hobby, and one goal you have for learning Python — each on its own line, using three separate `print()` calls.

## Exercise 3 — Print with multiple arguments (Medium)

File: `exercise_3.py`

Use a single `print()` call with multiple comma-separated arguments to print the words `"Python"`, `"is"`, and `"fun"` so they appear on one line separated by spaces.

## Exercise 4 — Basic arithmetic output (Medium)

File: `exercise_4.py`

Print the result of `12 + 8`, then on a new line print the result of `12 * 8`, using `print()` directly on the expressions (no variables needed).

## Exercise 5 — Mini bio script (Hard)

File: `exercise_5.py`

Write a script that prints a small formatted "bio card" using only `print()`, for example:

```
********************
Name: Ada
Skill: Python (Day 1)
********************
```

Use at least one line made entirely of a repeated symbol, and at least four total `print()` calls.
